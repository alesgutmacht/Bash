**Для ускорения распространения файлов на соседние хосты можно в цикле выполнять операцию копирования**  
**Также в цикле можно выполнять одинаковые команды**  

*Диапазон адресов 1*  
```
192.168.1.{2..4}
192.168.1.2, 192.168.1.3, 192.168.1.4
```
*Диапазон адресов 2*
```
192.168.1.{7..9}
192.168.1.7, 192.168.1.8, 192.168.1.9
```
  
*Копирование на соседние хосты:*  
```
Копируем необходимый файл на 1 хост (192.168.1.2), далее в цикле файл копируется на остальные хосты
for i in 192.168.1.{2..4} 192.168.1.{7..9}; do scp -p /home/user/file1.txt $i:/home/user/; done
```
*Выполнение на соседних хостах:*  
```
Содержимое скопированных файлов отображается на экране
for i in 192.168.1.{2..4} 192.168.1.{7..9}; do ssh $i 'hostname; cat /home/user/file1.txt'; done
```
